The tests are configurable with these system properties:

See
* https://bonigarcia.dev/webdrivermanager/
* https://bonigarcia.dev/selenium-jupiter/

| Property           | Description                                                                                      |
|--------------------|--------------------------------------------------------------------------------------------------|
| training.baseUrl   | default = http://localhost:8888/training                                                         |
| wdm.defaultBrowser | default = CHROME<br/>possible values = CHROME, FIREFOX, OPERA, EDGE, IEXPLORER, CHROMIUM, SAFARI |
